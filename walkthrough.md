## 一、交互式哲学知识卡片系统

### 1.1 知识卡片（18 张）

| 类别 | 数量 | 示例概念 |
|------|------|----------|
| 辩证法 | 6 | 对立统一、必然与偶然、量变与质变、现象与本质、主要/次要矛盾、自由与必然 |
| 认识论 | 3 | 实践与认识、感性与理性、已知与未知 |
| 价值论 | 2 | 内在价值与外在价值、时间维度与价值判断 |
| 伦理学 | 1 | 个人与社会责任 |
| 其他哲学 | 3 | 文化身份与传承、生存状态反思、时空维度 |
| 思维模型 | 3 | 辩证思维、系统思维、批判性思维 |

**每张卡片包含：**
- **正面**：标题（中英）、分类标签、哲学概述、历年考查年份标签、考查频率评级（1-5 星）
- **背面**（点击翻转）：深度解释、真题考查实例、写作建议、关联思维模型

### 1.2 分类导航筛选
7 个分类按钮（全部 / 辩证法 / 认识论 / 价值论 / 伦理学 / 其他 / 思维模型），点击即筛选，带入场动画。

### 1.3 25 年命题时间线
底部展示 2000-2025 共 26 年上海高考作文题目及其哲学标注，每项带分类色标标签。

## 二、AI Agent 工作台（核心功能）

### 2.1 审题分析引擎 [analyzeEssayTopic()](file:///d:/Vibe%20coding/gaokao-agent-main/app.js#777-840)

输入作文题目后自动输出：

| 模块 | 说明 |
|------|------|
| **题型检测** | 自动判别 4 种题型：关系辩证型 / 价值判断型 / 方法路径型 / 现象思辨型 |
| **题眼提取** | 中英双语关键词提取（正则 + 分词） |
| **哲学分类匹配** | 7 条规则引擎，自动匹配辩证法/认识论/价值论/伦理学/思维模型 |
| **审题切口建议** | 最多 4 条具体切入角度 |
| **概念界定** | 自动为题眼词生成界定建议 |
| **三种立场可选** | 条件性肯定 / 条件性质疑 / 关系整合，各带风险提示 |
| **核心立意** | 一句话中心论点 |
| **三段式论证骨架** | 按哲学类别自动生成段落任务 |
| **隐含前提 & 边界检查** | 阅卷隐含标准提醒 |
| **高分避坑（防跑题）** | 常见跑题模式警告 |
| **推荐知识卡片** | 按匹配度排序，点击可联动筛选 |
| **考场可写句** | 开头句 / 中心句 / 结尾句，可直接改写 |

### 2.2 防跑题诊断 [runOffTopicCheck()](file:///d:/Vibe%20coding/gaokao-agent-main/app.js#1305-1418)

| 指标 | 算法 |
|------|------|
| **题眼覆盖度** | 题眼词在草稿中出现比例 |
| **语义贴合度** | 同义词扩展后计算段落与题目语义重合 |
| **段落级定位** | 逐段诊断：贴合度百分比 + 问题列表 + 改写建议 |
| **题眼-段落对齐图** | 可视化每段命中和缺失的题眼词 |
| **反向约束检查** | 关系题缺转折？价值题缺条件？方法题缺路径？自动扣分 |
| **论证漏洞扫描** | 5 类：结论绝对化 / 例证堆砌 / 结论跳跃 / 单边论证 / 概念漂移 |
| **一键修复** | 每个漏洞可一键改写段首句，自动更新草稿并重跑诊断 |
| **风险等级** | 高 / 中 / 低，带扣题指数（0-100） |

### 2.3 维度评分系统 [scoreEssayDraft()](file:///d:/Vibe%20coding/gaokao-agent-main/app.js#1419-1489)

| 维度 | 满分 | 评价内容 |
|------|------|----------|
| 审题立意 | 20 | 基于防跑题扣题指数 |
| 结构章法 | 20 | 段落数、结构标志词、句群完整性 |
| 论证与材料 | 20 | 因果推理密度、例证数、辩证信号 |
| 语言表达 | 20 | 字词量、句式多样性、重复检测 |
| 思辨深度 | 20 | 辩证层数、多视角覆盖、推理链深度 |

**附加输出：**
- 总分 / 70分折算 / 上海分档（一类上 → 五类卷，共 11 档）
- 分档对标：阅卷评语 + 冲档目标 + 冲档动作
- 优先提分动作列表

### 2.4 考场模式（25 分钟 / 800 字）

| 功能 | 说明 |
|------|------|
| **倒计时器** | 25:00 → 00:00，支持暂停/继续/重置 |
| **实时字数统计** | 实时显示 `当前/800`，超限变红 |
| **节奏提醒** | 在 24/20/15/10/5/2 分钟节点推送写作策略提醒 |
| **动态防跑题快检** | 剩余 10/5 分钟时自动检测偏题风险 |
| **字数提醒** | 接近 760 字、超 800 字时推送提醒 |
| **最后 3 分钟冲刺面板** | 自动生成 3 条可执行冲刺动作按钮 |
| **一键执行** | 点击冲刺动作可自动定位段落、高亮选区、标出冗余句 |

### 2.5 范文库 & 示范题卡

| 功能 | 说明 |
|------|------|
| **内置范文** | 2 篇考场风格范文（论证型 + 价值判断型） |
| **分类筛选** | 全部 / 收藏 / 问题式命题 / 关系辩证题 / 价值判断题 |
| **收藏** | 星标收藏，LocalStorage 持久化 |
| **一键加载** | 点击加载范文到题目 + 草稿区 |
| **示范题卡** | 预置"综合与创新"完整高分骨架 + 低分跑题示例 |
| **评分基准** | 同题三档（高分/中档/跑题）评分对照表 |

### 2.6 提纲生成

点击"一键生成三段提纲草稿"，自动按题型生成 3 段 × 2-3 句的结构化提纲，同步写入草稿区。

### 2.7 辅助功能

- **填充示例**：随机填充 3 个预设题目 + 配套草稿
- **复制结果**：一键复制分析报告文本到剪贴板
- **知识卡联动**：分析结果中推荐的卡片可点击跳转到卡片区并自动筛选

## 三、视觉与交互设计

| 特性 | 实现方式 |
|------|----------|
| 暗色主题 | CSS 自定义属性，深蓝色系配 glassmorphism |
