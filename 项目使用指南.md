# 上海高考作文 AI Agent 项目使用指南

## 1. 项目定位
这是一个面向上海高考作文训练的前端工作台，核心目标是：

- 提升审题与立意准确度
- 强化思辨深度与论证闭环
- 降低跑题风险
- 支持考场节奏化训练（25分钟/800字）

当前版本包含：题目分析、三路径训练、逐步解锁训练、防跑题检查、评分报告、范文生成与素材池等功能。

## 3. 功能总览（你会用到的主流程）

### 3.1 分析题目
入口按钮：`分析题目`

会输出：

- 出题人意图
- 三步拆题法
- 必答清单
- 思辨三维训练包
- 上海一类卷冲刺卡
- 三路径深度思辨训练
- 逐步训练模式（可解锁）

### 3.2 草稿相关
- `一键转草稿骨架`：根据分析结果生成可续写骨架
- `一键生成训练提纲卡`：生成三维训练提纲
- `一键写入考前清单`：写入一类卷冲刺清单
- `一键写入三路径草稿`：写入三路径训练草稿

### 3.3 模板句快速插入
在三维训练包中，每条模板句可：

- 插入第1段
- 插入第2段
- 插入第3段

### 3.4 防跑题与评分
- `防跑题检查`：输出风险等级、三维缺口、漏洞扫描
- `一键补缺`：自动补句并高亮新增句
- `草稿评分`：按上海维度评分（总分/折算/分档）
- `一键提分改写`：自动执行提分动作

### 3.5 范文生成
按钮：`一键生成800-850字范文`

支持案例素材池：

- 自动匹配
- 科技创新
- 教育成长
- 文化传播
- 社会治理

## 4. 推荐训练流程（每天 1 题）

1. 粘贴题目，点击 `分析题目`
2. 进入 `逐步训练模式`：
   - 先做第1步（概念辨析）
   - 达标后解锁第2步（分类讨论）
   - 达标后解锁第3步（现实关联）
3. 点击 `防跑题检查`，再执行 `一键补缺`
4. 点击 `草稿评分`，看薄弱维度
5. 必要时点 `一键提分改写`
6. 最后开 `考场模式` 25分钟重写一版

---

## 5. 逐步训练模式说明

逐步训练是本项目长期提分核心，特点：

- 按“题目”维度保存进度（本地存储）
- 每步先写再检，达标后自动解锁下一步
- 可 `重置本题进度` 反复训练

三步目标：

1. 概念辨析：写清“不是……而是……”与边界
2. 分类讨论：写出基础分类 + 次级分类 + 比较分析
3. 现实关联：形成“问题-分析-对策”闭环

## 6. 常见问题与排查

### 6.1 页面打不开
- 确认命令窗口仍在运行 `python -m http.server`
- 访问 `http://127.0.0.1:5500`（不是 `3000`）

### 6.2 功能按钮点击无反应
- 先 `Ctrl + F5`
- 检查是否输入了题目
- 点击 `基础自检` 查看失败项

### 6.3 生成内容看起来重复
- 切换案例素材池
- 先运行 `分析题目` 后再生成
- 用 `一键补缺` + `一键提分改写` 做二次优化

### 6.4 本地进度丢失
- 进度使用浏览器 `localStorage`
- 清理浏览器缓存/隐私数据后会丢失

2. `分析题目` 有输出
3. 模板句可插入第1/2/3段
4. `防跑题检查` 可运行
5. `一键补缺` 后草稿自动高亮新增句
6. `一键生成800-850字范文` 可运行
7. 切换不同案例素材池，生成例证有变化
8. 逐步训练第1步达标后可解锁第2步

## 8. 给老师/同学的使用建议

- 课堂：先用“分析题目 + 三路径训练”做共同审题
- 课后：学生按逐步训练单独打卡
- 讲评：用“防跑题 + 评分报告”做针对性反馈
- 冲刺：考前使用“一类卷冲刺卡 + 考场模式”定型
