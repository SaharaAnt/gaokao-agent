【主持人语】

如 何 论 证

——必修下第八单元写作任务指导

上海同济二附中 江练

         

统编教材高中语文必修（下）第八单元主题为“倾听理性的声音”，属于“思辨性阅读与表达”任务群。单元选文有奏疏、书信、辞赋、史论等，文体多样，从不同角度展现了古人面对国家社会问题时的理性思考。所谓“倾听”，是指理解、领悟乃至鉴赏、运用。因此单元教学目标旨在引导学生领会“作者观点及其现实针对性”，把握其“解决现实问题的理性思维方式”，感悟古代士人的“责任与担当”。同时鉴赏其说理艺术，单元领悟其论事说理的言说能力，养成审辨质疑的思维习惯，并能够运用到自己的写作中。

单元写作任务如下：

有人认为，道德底线低的人有时比道德底线高的人占优势，你做不出来的，他做得出来，所以他们总能如鱼得水。但师长告诫我们：要坚持“高线”，守住“底线”；只有守住“底线”，才是守住了为人的根本。以“底线”为话题写一篇不少于800字的文章，题目自拟。

         

一、明确写作要求：基于任务群目标的写作要点

课标“思辨性阅读与表达”任务群的学习目标与内容提出，学生要“学习表达和阐发自己的观点，力求立论正确，语言准确，论据恰当，讲究逻辑。”本单元要落实的写作要点是学习“如何论证”。单元学习任务后附了“如何论证”的写作知识短文，提供了写作策略性知识。在此之前，学生已经有过“写作针对性”“如何阐述观点”等写作要点的训练经历。学生习作暴露出的主要问题是论点缺乏阐释，堆砌论据，论证方法单一。对论点中包含的关键概念不做阐释，喜欢用例证法但对论据与论点关系不做分析。基于以上分析，本次写作任务的训练要求是：指导学生在已有的写作知识积累与本单元阅读积累的基础上，学会阐释论点，选择论据，完善论证过程，提升议论文写作的分析能力。

二、实施写作过程：读写融合为策略的活动指导

活动一：温故知新，储备写作知识

必修下阶段，学生对议论文写作三要素不陌生，但是大多还停留在“知道”的层面，对三要素的具体要求不甚明了。因此指导学生自主阅读单元写作知识短文，通过梳理“如何论证”相关的写作知识，可以加深学生对议论文写作知识的认知理解。梳理表格如下：

学生阅读知识短文，理解“论点”“论据”“论证”等基本概念内涵，自主梳理提炼出论证的要求，比如论点的确立、论据的选择、结构安排等等。学生自主阅读梳理相关写作知识，明确写作三要素的具体要求，为本次写作做好知识储备。

活动二：读写融合，理解“理性表达”共性要求。  

单元学习任务中有一项读写结合的任务——“理性的声音需要理性的表达，这就要求观点鲜明，言必有据，运用合适的论证方法（如比喻和类比）等。选取本单元一篇课文，从理性表达的角度进行分析，写一篇短论”。

我们可以依托这项学习任务，指导学生梳理单元课文相关论证知识，关注不同文本在理性表达上的特点，将文本阅读获得的感性认识，与理性表达的基本要求统一起来，形成对“如何论证”的初步认识，为落实单元写作任务打下基础。可以请同学们对本单元的论述文进行如下梳理。

表格梳理项目包含议论文写作的几大要素：写作针对性，确立观点，论证方法，论据选择等。通过对本单元四篇经典文章论证特点的梳理，学生对怎样的论证过程是有说服力的，如何在议论文写作中做到理性表达有更清晰直观的认识。进而可以借鉴经典文言作品中论证观点的方法，学习以理性的方式探讨问题的思维方式，并在独立写作中加以迁移运用。

活动三：读题审题，分析具体要求

与学生常见的材料作文有所不同，这是一道给材料的“话题”作文。话题作文写作自由度比材料作文高，但也不能无视命题人给出的材料范围与隐含倾向。话题作文的材料往往对写作范围和主旨起到提示和限制的作用。首先它限定了“底线”的外延，要求写作者讨论的是“道德底线”，而非其他什么底线。其次，它提示了写作者思考的切入点。材料中呈现了两种人对道德底线的不同看法，写作时可以从这两种矛盾的观点切入，探讨孰是孰非及背后的原因。再者，材料的表述也隐含了命题者的立意倾向。有人的观点代表一部分无视社会底线的人群的价值观，而师长的告诫则代表着一个健康社会对“道德底线”应有的价值判断。当然，材料中的观点只是为写作提供某种启发或参考，不一定完全按照师长所说，可以从现实分析出发，提出自己的看法。

活动四：构思作文，完成写作提纲

从以上分析来看，话题核心、写作范围，以及主旨立意都可以从分析材料中获得。下一步就是构思作文，教师可提供写作构思任务单，表格如下。

         

|   |   |   |
|---|---|---|
|我确定的观点是||各层次的关系；论证方法|
|需要阐释的核心概念||
|针对何种现象||
|阐述层次1——<br><br>分论点及论据|||
|阐述层次2<br><br>分论点及论据|||
|阐述层次3<br><br>分论点及论据|||
|我的结论|||

构思作文首先要明确论点，还要想清楚为了阐明观点，我的文章打算写几个层次？哪些论据可用，采用何种论证方法，各层次之间是什么逻辑关系等等。展开论证之前，不能忽视对观点中包含的核心概念进行阐释。比如什么叫道德底线？什么叫道德底线低？一般认为社会道德底线是民众共识的统一规范，为何还存在道德底线低？难道说道德底线是因人而异忽高忽低的吗？显然这里所说的“道德底线低”是指个人的道德底线，而不是社会道德底线。区分清楚这两个概念的差异，下面的论述才不会出现概念不统一的情况。此外，所谓的“道德高线”、“如鱼得水”需要明确其内涵或所指现象。文章的话题是“底线”，材料中出现了“道德高线”，这两者主次要分清。辨明观点中的核心概念是为下文论证打基础的工作。明确文章所持观点，才能选择恰当的论据，论据的解说与分析也就不容易跑偏了。

         

【学生佳作】

守住底线之本 坚持高线之准

同济二附中 高二7班 甘之轩

![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/RV6ZOY8ib7azza9UlDlwEz7MW809iaxkdSqGIjvfUibWZUl63p02WU9MxUwpYxiaKQKaib0uMa4urcGJCcjgtIhjVtQ/640?wx_fmt=jpeg&tp=wxpic&wxfrom=5&wx_lazy=1#imgIndex=3)

有人认为，道德底线低的人有时比道德底线高的人占优势，你做不出来的，他做得出来，所以他们总能如鱼得水。何谓道德底线？奉持低道德底线的人果真总能如鱼得水吗？（开篇入题，引用了材料中有人说的内容，并抛出了问题，奉持低道德底线的人果真总能如鱼得水吗？先把批驳的靶子竖起来，接下来要驳斥这种错误的看法。这是先驳后立的写法）

道德底线，是指人们应该遵循的社会道德最低警戒线。个人道德是一种高低都不封顶的存在，而社会道德却具有边界。因此，有人所说的“道德底线低”，应该是指个人的道德底线低于社会道德底线。（作者没有立刻批驳有人的看法，而是先界定重要概念的内涵，如何为道德底线，个人道德与社会道德的差别，并在此基础上阐释了所谓的如鱼得水的意思——是放松自我道德要求，突破底线谋取利益。这是明确要讨论的话题核心，避免歧义和漏洞，是全文立论的重要基础。）所谓“如鱼得水”，不过是个体放松自我道德要求，不惜突破社会底线而谋取利益。人性有趋利避害的本能，在功利主义思潮的驱动下，人们更是罔顾道德底线，理直气壮地追逐利益。这些人往往自诩聪明，认为自己善于变通，左右逢源，如鱼得水。殊不知在道德底线边缘打擦边球甚至突破底线而获得的利益，犹如空中楼阁，只能让人获得短暂的快慰，最终会在某一天訇然倒塌。（接下来简明而要剖析有人这番话背后的原因是人性趋利避害和功利主义思潮使然。并揭示坚持低道德底线的人的真实下场，从而证明奉持低道德底线的人并不能如鱼得水这一观点）

揆诸当下，奉持低道德底线的人不在少数。自媒体时代，为了博取流量和眼球，故意夸大扭曲地报道事实，误导受众埋没真相的新闻比比皆是。今年5月份，戏曲名家何赛飞参加某戏剧节目评委时，针对行业现状的一番含泪发言振聋发聩，引起了极大地反响和共鸣，迅速冲上热搜。可是某些营销号在传播视频时，为了博取眼球断章取义，甚至将标题取名为“何赛飞痛斥梅花奖”，严重歪曲了她的本意。短时间内来看，获取的流量可能让营销号获得一定的利益。但从长远来看，歪曲事实的报道不仅混淆视听，对社会价值观造成负面影响，也必将使其自身失去流量与利益。可见，降低道德底线并不能使人如鱼得水，坚持底线才是个人安身立命的根本，也是社会和谐稳定的基石。（本段联系现实，列举事实驳斥有人的错误认识：并提出“坚持底线才是个人安身立命的根本，也是社会和谐稳定的基石”的观点。论据来自新闻报道，准确可靠，能围绕论点对论据进行解说与分析，具有较强的说服力。）

北岛说“卑鄙是卑鄙者的通行证，高尚是高尚者的墓志铭”，当一些人奉持低道德底线而如鱼得水的时候，另一群人却用生命筑起道德高线。手持旌节与风雪冷月作伴的苏武，遵守心中的国家大义坚守多年，不为劝降的想法和高官俸禄所动，感动了一代又一代人。扎根山区的张桂梅校长，甘愿化作大山里的一盏明灯，指引无数贫困学子通过读书之路走出大山，改变命运。她的事迹成为当代人们心中的道德典范。他们孜孜以求“止于至善”的精神境界成就了人类的道德高线。或许有人会说，道德高线能有几人能够达到？康德有言：“在这个世界上，有两样东西值得我们仰望终生：一是我们头顶上璀璨的星空，二是人们心中高尚的道德法则。”要知道正是这些坚持“高线”的典范，如明星般闪耀在历史的天空，照亮了人类文明前进的道路。物欲横流的时代，我们更需要这样的精神引领，方能真正守住底线而不动摇。（本段转入论述道德高线的价值，列举了苏武和张桂梅两则事例，并引用康德名言，强调道德高线引领对人类精神文明进步的意义，从而论证人类需要道德高线。）

曾子曰，吾日三省乎吾身。道德本质上是一种向内自省的力量，唯有不断磨砺个人私德，我们才能在喧嚣浮华中守住做人的根本，捍卫住心中的道德底线，并且不断地向道德高线靠近。愿我们都能守住底线之本，坚持高线之准，即便身处万丈波涛，仍能一苇杭之。（从论述思路上来看，本段是在上文分析问题的基础上，提出解决问题的办法，即不断内省，磨砺私德，守住底线，并不断向高线靠近。最后重申论点。）

         

总 评：

这篇习作首先引述题目材料中有人的错误思想，并提出质疑。接下来没有立即展开反驳，而是先界定概念，厘清了个人道德底线与社会道德底线的关系，再指出个人道德底线低于社会道德底线的人“如鱼得水”只是暂时的。接着联系现实举自媒体为了博眼球而歪曲新闻事实的例子，从个体长期发展乃至社会稳定和谐两方面，论证了坚守底线的必要性与重要性。文章没有到此结束，而是推进一层，从底线谈及高线，强调物欲横流的当下，人类需要道德高线的引领，方能真正守住底线。文章紧扣话题核心，观点清晰，有理有据，论证严密，显示出小作者良好的思辨能力，和对社会热点问题敏锐关切的人文情怀。（江 练）
